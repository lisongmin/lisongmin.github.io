c++

  • 2020-04-02 阅读了来自 Sutter’s Mill 的 GotW #100: Compilation Firewalls
  • 2019-06-02 阅读了来自 Mark Isaacson 的 memcpy, memmove, and memset are obsolete!
  • 2019-06-27 阅读了来自 Eli Bendersky 的 SFINAE and enable_if 学会了enable_if的用法
    • enable_if可以用于模板参数中
      template <class T,
         typename std::enable_if_t<std::is_integral<T>::value>* = nullptr>
void do_stuff(T& t) {
    // an implementation for integral types (int, char, unsigned, etc.)
}
    • 可以用于函数的返回类型中
      template <class T>
typename std::enable_if<std::is_arithmetic<T>, bool>::type
signbit(T x)
{
    // implementation
}
    • 还可以用于模板函数的入参中，这个之前没有见过
      template <class _InputIterator>
vector(_InputIterator __first,
       typename enable_if<__is_input_iterator<_InputIterator>::value &&
                          !__is_forward_iterator<_InputIterator>::value &&
                          ... more conditions ...
                          _InputIterator>::type __last);
